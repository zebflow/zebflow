# Zebflow Learning Path

Use this guide in order if you are new to Zebflow:

1. Understand the platform and what a project contains.
2. Learn how pipelines and templates combine into one app.
3. See a simple blog structure so the repo shape becomes concrete.
4. Learn the frontend stack: React + Zeb libraries.
5. Browse examples, project management, databases, hub, and agentic tooling.

Zebflow is opinionated:

- pipelines define behavior and entry points
- templates define UI and browser interactivity
- both live in one project and are activated together
- internal libraries under `zeb/*` are the preferred frontend surface

The rest of this help tree is arranged around that flow.

# Gallery + Source Examples

Zebflow should be learned from real working shapes, not only API reference.

## Use cases you should expect

- blogs and editorial systems
- dashboards and admin panels
- realtime experiences
- agentic scheduling and automation
- scraping + ingestion flows
- geospatial / map apps
- internal tools

## Where to look

- `help("pipeline/examples")` for project-style pipeline recipes
- `help("web/deckgl")` for map-heavy UI
- `help("web/libraries")` for bundled frontend capabilities

The examples are meant to show both:

- what Zebflow can do
- what the source code and project shape look like
